# 케어링 랜딩 페이지 전환율 개선 제안

## 📊 현재 문제점 분석

### 1. 헤드라인의 문제점
```
"소득 재산 상관 없이 부모님 돌봄 국가지원 제공"
```
**문제점:**
- 너무 거창하고 추상적임
- "국가지원"이라는 단어가 절차가 복잡해 보일 수 있음
- 고객의 즉각적인 고통(Pain Point)을 건드리지 못함

### 2. CTA(Call to Action) 약함
```
"무료 상담 신청하기"
```
**문제점:**
- 소극적임
- 긴장감/희소성 부족
- 행동의 명확성 부족

### 3. 복잡한 폼
- 성함 + 간병 대상 + 연락처 + 메시지
- 인지 부하가 높음

### 4. 감정적 연결 부족
- 통계 중심의 논리적 접근
- 공감/불안 감정 자극 부족

---

## 🎯 개선 전략: AIDA 모델 적용

### Attention (주의) - 강력한 오프닝
### Interest (흥미) - 공감과 문제 인식
### Desire (욕구) - 해결책과 혜택
### Action (행동) - 명확한 CTA

## 🚀 구체적 개선안

### 1. 히어로 섹션 개선 (A/B 테스트 제안)

#### Option A: 고통(Pain) 중심
```html
<span class="hero-tag">부모님 치매, 우리 가족의 현실이 될 수 있습니다</span>
<h1>치매 간병비 <span class="point">월 200만원</span>,<br>준비 안 된 가족은 망합니다.</h1>
<p>65세 이상 10명 중 1명은 치매를 경험합니다.<br>국가 지원 제도로 간병비 부담을 85%~100% 줄이세요.</p>
```

#### Option B: 해결(Solution) 중심
```html
<span class="hero-tag">소득·재산 상관없이 국가 지원받으세요</span>
<h1>부모님 돌봄 <span class="point">국가 비용</span>으로<br>간병비 0원에 가깝게 만드는 법</h1>
<p>복잡한 장기요양등급 신청, 전문가가 1:1로 무료 상담해 드립니다.<br>지금 바로 확인하세요.</p>
```

#### Option C: 질문(Question) 중심
```html
<span class="hero-tag">부모님 돌봄, 어떻게 준비하고 계신가요?</span>
<h1>치매 진단 후 <span class="point">간병비 월 200만원</span>,<br>감당할 수 있으신가요?</h1>
<p>국가 지원 제도를 활용하면 간병비의 85%~100%를 지원받을 수 있습니다.<br>복잡한 절차는 저희가 도와드립니다.</p>
```

### 3. 폼 간소화

#### 현재 폼 구조
```
1. 성함 *
2. 간병 대상 *
3. 연락처 *
4. 메시지 (선택)
```

#### 개선안: 2단계 폼
**Step 1 (간소화)**
```
연락처 *
```
→ "010-0000-0000" 형식 자동 변환

**Step 2 (상담 후)**
```
성함, 간병 대상, 상세 내용
```
→ 전화 상담 시 수집

**코드 예시:**
```html
<!-- 1단계: 연락처만 받기 -->
<div class="form-card reveal">
    <div class="section-title">
        <h2>부모님 간병비 국가 지원,<br>지금 바로 확인하세요</h2>
        <p>연락처를 남기시면 전문 컨설턴트가 1시간 이내에 연락드립니다.</p>
    </div>
    <form id="consultForm">
        <div class="form-group">
            <label>연락처 *</label>
            <input type="tel" id="userPhone" class="form-control"
                placeholder="010-0000-0000" maxlength="13" required>
            <span class="error-msg">올바른 연락처 형식이 아닙니다.</span>
        </div>
        <div class="form-check-group">
            <div class="form-check">
                <input type="checkbox" id="agree" required>
                <label for="agree">개인정보 수집 및 이용 동의 (필수)</label>
            </div>
        </div>
        <button type="submit" class="submit-btn">
            지금 바로 무료 상담 신청
            <span style="font-size: 0.85rem; display: block; font-weight: 600;">
                ✓ 1시간 이내 연락
            </span>
        </button>
    </form>
</div>
```

### 5. 긴급성 & 희소성 요소 추가

#### 카운트다운 타이머
```html
<div class="urgency-box">
    <p>🔥 이번 주 <strong>무료 상담 3자리</strong> 남았습니다</p>
    <div class="countdown">
        <span class="time-box">02</span> :
        <span class="time-box">45</span> :
        <span class="time-box">30</span>
    </div>
    <p style="font-size: 0.85rem; opacity: 0.7;">시간 내에 신청하세요</p>
</div>
```

**JavaScript 구현:**
```javascript
// 카운트다운 타이머
function startCountdown() {
    const endTime = new Date();
    endTime.setHours(23, 59, 59); // 오늘 23:59:59까지

    const timer = setInterval(() => {
        const now = new Date();
        const diff = endTime - now;

        if (diff <= 0) {
            clearInterval(timer);
            return;
        }

        const hours = Math.floor(diff / (1000 * 60 * 60));
        const minutes = Math.floor((diff % (1000 * 60 * 60)) / (1000 * 60));
        const seconds = Math.floor((diff % (1000 * 60)) / 1000);

        document.querySelector('.time-box:nth-child(1)').textContent = String(hours).padStart(2, '0');
        document.querySelector('.time-box:nth-child(2)').textContent = String(minutes).padStart(2, '0');
        document.querySelector('.time-box:nth-child(3)').textContent = String(seconds).padStart(2, '0');
    }, 1000);
}
```

## 📈 A/B 테스트 우선순위

### 1순위: 히어로 섹션
- **A**: 현재 버전
- **B**: 고통 중심 메시지
- **측정指标**: 바운스률, 체류 시간

### 2순위: CTA 버튼
- **A**: "무료 상담 신청하기"
- **B**: "지금 바로 무료 상담 받기" (긴급성 표시)
- **측정指标**: 클릭률, 전환율

### 3순위: 폼 구조
- **A**: 4항목 폼
- **B**: 연락처 1개 항목 폼
- **측정指标**: 폼 완료율, 전환율
